## Setup

1. Install nodejs. If you are using ubuntu, you can install with package manager

``` sh
curl -sL https://deb.nodesource.com/setup_13.x | sudo -E bash -
sudo apt-get install -y nodejs
```

2. Run under project folder

```bash
npm install
```
